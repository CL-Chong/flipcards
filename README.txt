flipcards is a program that generates maths question-answer pairs using sympy.
The level of maths questions is suitable for UK A-Level (16-18) students.

How to use:
1. Run main.py and follow the instructions on the terminal to select questions
2. An interactive pop-up window will be opened. Left-click to reveal answer/see next question and right-click to exit.

Required packages: numpy, sympy, matplotlib

Currently implemented maths questions:
Differentiation:
    Basic Differentiation
    Chain Rule
    Product Rule
    Quotient Rule

Future plans:
Implement other question types such as simultaneous equations, integration, Taylor series
